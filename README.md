## 🕹️ Descripción

Este proyecto consiste en el diseño de la **interfaz gráfica de la página principal (Home)** de un portal de videojuegos.  
La aplicación ha sido construida utilizando las herramientas visuales de **Visual Studio**, aplicando principios de **diseño, estructura, alineación y coherencia visual**.

El objetivo es simular la apariencia de un sitio web de videojuegos, dividiendo la ventana principal en zonas bien diferenciadas:
- **Encabezado superior:** incluye el título y el menú principal (Tienda, Comunidad, Soporte, Perfil).  
- **Panel lateral:** contiene accesos directos a secciones como “Destacados”, “Mis juegos” y “Configuración”.  
- **Zona central:** muestra los juegos disponibles o destacados mediante paneles de contenido.
